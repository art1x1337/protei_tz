# PROTEI Landing 2026

Тестовое задание на позицию Frontend Developer.

Проект представляет собой адаптивный одностраничный лендинг с блоком программы лектория, выбором лекций и формой регистрации, выполненный на React, TypeScript, Tailwind CSS и React Hook Form.

## Стек

- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Hook Form

## Реализовано

- адаптивный одностраничный лендинг
- секция программы лектория с аккордеонами по дням
- выбор лекций
- счётчик выбранных лекций
- форма регистрации на лекторий
- валидация обязательных полей
- валидация email
- валидация телефона
- вывод сообщений об ошибках под полями
- модальное окно успешной отправки
- модальное окно ошибки
- ссылка на политику обработки персональных данных

## Локальный запуск

Установить зависимости:

```bash
npm install
```

Запустить проект в режиме разработки:

```bash
npm run dev
```

После запуска приложение будет доступно по адресу, который покажет Vite в терминале, обычно:

```bash
http://localhost:5173
```

## Сборка проекта

Собрать production-версию:

```bash
npm run build
```

Локально проверить production-сборку:

```bash
npm run preview
```

## Структура проекта

- `src/components` — UI-компоненты и секции лендинга
- `src/data` — данные для отображения лекций и секций
- `src/types` — TypeScript-типы
- `src/utils` — вспомогательные функции
- `public/privacy-policy.html` — страница политики обработки персональных данных
