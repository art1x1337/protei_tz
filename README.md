# PROTEI Landing 2026

Стартовый проект под тестовое задание для Frontend Developer.

Собрано на:
- React 19
- TypeScript
- Tailwind CSS
- React Hook Form
- Vite

## Что уже есть

- длинный одностраничный лендинг
- hero-блок
- блок преимуществ
- блок форматов участия
- программа круглого стола
- форма регистрации на круглый стол
- программа лектория с аккордеонами по дням
- выбор лекций по клику на карточки
- счётчик `Выбрано N лекций`
- форма регистрации на лекторий на React Hook Form
- валидация обязательных полей
- валидация email
- форматирование и валидация телефона
- success modal
- error modal
- мобильная адаптация

## Как запустить

```bash
npm install
npm run dev
```

Открыть в браузере:

```bash
http://localhost:5173
```

## Сборка

```bash
npm run build
npm run preview
```

## Быстрая проверка error modal

По умолчанию ошибка иногда появляется случайно при отправке формы.

Чтобы вызвать её гарантированно, введи в поле email значение со словом `error`, например:

```txt
error@test.ru
```
